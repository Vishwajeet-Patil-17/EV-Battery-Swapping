EV Battery Swapping Educational Website

A responsive and interactive educational website designed to explain the concept of electric vehicle (EV) battery swapping and its benefits. The project presents key information in a simple, user-friendly format with visual content and basic interactivity.

Explains the concept of battery swapping and its real-world importance
Highlights benefits like reduced downtime, lower infrastructure load, and cost efficiency
Includes a section on working process with an embedded video for better understanding
Covers different types of battery swapping methods (manual, automated, mobile)
Provides an interactive quiz to engage users and test understanding
Built using HTML, CSS, and JavaScript with a clean and responsive UI

This project demonstrates front-end development skills including layout design, responsive styling.
